# MigraNova — Sistema de Gestión de Extranjería

Aplicación web para despachos especializados en extranjería española.

## Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Página de inicio con acceso a los dos portales |
| `migranova-despacho.html` | App del despacho (expedientes, clientes, presupuestos, VeriFactu) |
| `migranova-cliente.html` | Portal del cliente (subida de documentación, chat) |

## Tecnología

- Frontend: HTML + CSS + JavaScript puro
- Base de datos: Supabase (PostgreSQL)
- Autenticación: Supabase Auth
- Almacenamiento de documentos: Supabase Storage

## Normativa

- RD 1007/2023 (VeriFactu / Requisitos SIF)
- Orden HAC/1177/2024
- RD 1155/2024 (Reglamento de Extranjería)
- LO 4/2000 (Ley de Extranjería)
